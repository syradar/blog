# Syradar Blog

Curated references and notes for web development topics, organized as docs and weekly link collections.

## Language

**Link Week**:
A weekly collection of curated links published as one issue.
_Avoid_: weekly dump, feed

**Links Explorer**:
The searchable all-links index at `/blog/links/` for cross-issue discovery.
_Avoid_: hub, stream

**Weeks Archive**:
The year-grouped index of Link Weeks used to browse all issues.
_Avoid_: timeline, stream

**Year Archive**:
A filtered view of Link Weeks for exactly one year.
_Avoid_: season, period view

## Relationships

- A **Weeks Archive** contains one or more **Year Archive** sections.
- A **Year Archive** contains one or more **Link Week** issues.
- A **Links Explorer** contains individual links from one or more **Link Week** issues.

## Example dialogue

> **Dev:** "Should this be in the **Weeks Archive** or a **Year Archive** page?"
> **Domain expert:** "Use **Year Archive** for focused browsing; keep **Weeks Archive** for cross-year discovery."

## Flagged ambiguities

- "weeks page" was ambiguous between all years and one year; resolved as **Weeks Archive** (all years) vs **Year Archive** (single year).
